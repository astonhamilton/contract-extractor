Target Artifact Domain:
- Question answered: what prior artifact does this change appear to affect?
- target_artifact.answer should be short prose describing the prior agreement, contract, renewal, SOW, work order, or related artifact this change appears to modify.
- target_artifact.citations should cite the linkage language supporting that interpretation.

Decision steps:
1. Identify the operative change instrument first.
2. Find explicit language such as amend, modify, renew, extend, replace, revise, or supplement.
3. Decide what artifact that language points to.
4. Write an honest note even if the target is only partially identified.

Guardrails:
- Do not force exact target identifiers when only a fuzzy description is visible.
- Do not confuse the change document title with the target artifact being changed.
- If several references appear, prefer the one explicitly tied to the operative change language.
