# Question Type Index

Use the user's question to decide which question-type workflow applies.

Current question types:
- governing document identification
  - use when the user asks which document governs a relationship, which agreement is in force, or what currently controls the vendor relationship

When a question clearly matches one of these types, follow that type's workflow and apply its learnings.

If a question does not cleanly match a named type, still use the same broad-to-narrow discipline:
- choose a small candidate document set from the corpus reference
- use document notes before page retrieval when possible
- use page notes before raw page retrieval when possible
- use exact pages rather than broad retrieval whenever you can
